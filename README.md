## Getting Started
clone the project
```
git clone https://github.com/vuestorefront/nuxt3-magento-sdk-storefront.git
```
cd into project
```
cd nuxt3-magento-sdk-storefront
```
Install dependencies
```
yarn install
```
run app
```
yarn dev
```
